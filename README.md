# filipwolanski.com

The source to filipwolanski.com. To run a local copy, run:

```bash
git clone https://github.com/fwolanski/filipwolanski.com.git
cd filipwolanski.com
npm install
grunt
```

Open `http://localhost:9000` in your favorite browser.
